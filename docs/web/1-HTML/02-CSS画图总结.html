# CSS画图总结

# 1. 三角形

  • 宽高设为0
  • 四个边框设置border-width，border-style，border-color即可
  • 如果某个三角要变为透明，设置border-color为transparent

其他：

  • 直角三角形可以用两个边拼起来
  • 矩形可以是width不为0

# 2. 扇形

  • 画一个圆
  • 画另外两个半圆，绝对定位（使用border-radius）
  • 通过transform: rotate(90deg);旋转相应角度（transform-origin旋转基点）
